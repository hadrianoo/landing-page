# Landing Page Project

This project is a static landing page built as part of the curriculum from The Odin Project.

## Project Goal

The goal of this project was to:

- Practice semantic HTML structure
- Apply CSS styling and layout techniques
- Use Flexbox for positioning elements
- Follow a structured Git workflow
- Deploy the project using GitHub Pages

The focus was on building a clean layout based on a provided design mockup.

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Git & GitHub
- GitHub Pages (deployment)

## Live Preview

Live: https://hadrianoo.github.io/landing-page/

## What I Practiced

- Structuring a multi-section landing page
- Creating layout patterns
- Working with positioning
- Translating a static design into code
